#Интернет-магазин

- 2 интерфейса для продавцов и покупателей
- продавец:
добавить товар (описание, цена, фото?)
отслеживание наличия, цены, вывод денег за продажу
доставка?
акции?
безопасность?

общий обзор товаров
поиск товаров
категории?
добавление товаров в корзину
добавление товаров в избранное
отзывы (писать\читать)
